[ChapterNumber] Overview

CRISPR technology allows one to induce mutations in a given target site. When the target areas are amplified using amplicon sequencing, one can efficiently examine the target areas and analyze them bioinformatically in order to identify mutations and quantify them.




[ChapterNumber].1 Computational Analysis Workflow

Quality filtered amplicon sequencing reads were stitched together using NGmerge. Primer sequences within each read were used to assign it to a reference amplicon. Once assigned to a reference amplicon, the reads is aligned against the relevant gRNA sequences. The aligned region (along with flanking bases) is checked against the sequences within a reference wild-type sample, and according to this check is deemed either mutated or unmutated.
